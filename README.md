# Проект "Автосалон"

Этот проект представляет собой простую консольную систему для управления автомобилями, клиентами и заказами в автосалоне.

## Описание

Система позволяет:
- Искать автомобили по марке.
- Фильтровать клиентов по бюджету.

## Технологии
- Java
